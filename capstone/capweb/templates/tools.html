{# ==============> OVERVIEW <============== #}

Information is only useful if it's accessible.

The capstone of the Caselaw Access Project is a robust set of tools which facilitate access to the cases and their associated metadata. We currently offer two ways to access the data: our API, and bulk downloads.

{# ==============> API <============== #}

The API

Click for our API, and related documentation.

Our open-source API is the best option for anybody interested in programmatically accessing our metadata, full-text search, or individual cases.

{# ==============> BULK!! <============== #}

Bulk Downloads

Click here for bulk data downloads.

If you need a large collection of cases, you will probably be best served by our bulk data downloads. With no registration or formal agreements, we offer zip files containing all cases from Illinois and Arkansas, either in plain text, or marked up with XML. Non-commercial researchers who want a larger, or different subset of our cases should email us for more information. Those interested in licensing this data case for commercial use should contact Ravel Law for more information.
